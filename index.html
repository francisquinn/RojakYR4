API Documentation

api_endpoint/user/games/get/gameName

This returns a list of games with similar names

Parameter
gameNameThe name use to search for the game
userId[OPTIONAL] Required for users to view their own games that are set to private (i.e. games that can't appear in normal search).

api_endpoint/user/games/get/id

Search for a single specific game using object_id

Parameter
objectIdObjectt id of the game
userId[OPTIONAL] Required for users to view their own games that are set to private (i.e. games that can't appear in normal search).

api_endpoint/user/games/create

Create a game to begin record, returns the game's id

Parameter
teamIdTeam id of the user's team
userIdThe user id of the user who created this game
publicWhether this game is available to the public, 0 = private, 1 = public
gameNameThe name of the game

api_endpoint/user/game_events/create

Sends the text command to be parsed and convert into a game event, which is then updated to a game, returns a new possession id if the current possession ends

Parameter
objectIdObject id of the game, received when called user/games/create
pid[OPTIONAL] The possession id, starts from 0 if there is none
textInputThe text input converted from the user's voice input
timethe time stamp in seconds when the voice input is send out, starts from 0 when the game begins
