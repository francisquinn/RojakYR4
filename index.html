Create game

Creates a game using a given game id and start time

sample query

?command=create_game&gameID=100&startTime=1920



Create possession

Creates a possession event based on the given game ID and start time

?command=create_possession&gameID=100&possessionID=21&startTime=77&possessionTeam=42



update possession event

Pushes a possession event to a possession with the given possession ID

?command=update_possession_event&possessionID=21&eventID=1&timeInSecond=32&eventType=1&playerID=33&teamID=34&outcomeID=1&outcomeTeamID=34&outcomePlayerID=33
