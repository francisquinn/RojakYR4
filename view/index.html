API Documentation

api_endpoint/user/games/get/gameName

This returns a list of games with similar names

Parameter
gameNameThe name use to search for the game
userId[OPTIONAL] Required for users to view their own games that are set to private (i.e. games that can't appear in normal search).
Returns
resultAn array of game objects
codeStatus code of the operation

api_endpoint/user/games/get/id

Search for a single specific game using object_id

Parameter
objectIdObjectt id of the game
userId[OPTIONAL] Required for users to view their own games that are set to private (i.e. games that can't appear in normal search).
Returns
resultA game object with the corresponding object id
codeStatus code of the operation

api_endpoint/user/games/create

Create a game to begin record, returns the game's id, this action will also create a active_match document which is required for user/update

Parameter
teamIdTeam id of the user's team
userIdThe user id of the user who created this game
publicWhether this game is available to the public, 0 = private, 1 = public
gameNameThe name of the game
locationThe location of the game
matchDateDate of the game
matchTimeThe time when the game started
teamColorColor of the team
oppTeamColorColor of the opposite team
Returns
codeStatus code of the operation

api_endpoint/user/games/delete

Deletes a game by game's object id

Parameter
objectIdObject Id of the game
Returns
codeStatus code of the operation

api_endpoint/user/game_events/create

Sends the text command to be parsed and convert into a game event, which is then updated to a game, returns a new possession id if the current possession ends

Parameter
objectIdObject id of the game, received when called user/games/create
pid[OPTIONAL] The possession id, starts from 0 if there is none
textInputThe text input converted from the user's voice input
timethe time stamp in seconds when the voice input is send out, starts from 0 when the game begins
Returns
eventThe generated event
codeStatus code of the operation

api_endpoint/user/game_events/update

Updates a specific event by using the game id, possession id and game event id as reference

Parameter
objectIdObject id of the game
pidThe possession id
eidThe event id
Returns
codeStatus code of the operation

api_endpoint/user/register

Register a user

Parameter
usernameThe username of the user
passwordPassword of the user.
emailEmail of the user.
Returns
userIDUser id to initiate a login session
codeStatus code of the operation

api_endpoint/user/register/checkNameDuplicates

Check whether a username is already registered, usernames are case sensitives

Parameter
usernameThe username of the user
Returns
result1 = this username has been registered, you cannot use it to register, 0 = this username is available
codeStatus code of the operation

api_endpoint/user/register/checkEmailDuplicates

Check whether an email is already registered, email are not case sensitives

Parameter
emailThe email of the user
Returns
result1 = this email has been registered, you cannot use it to register, 0 = this email is available
codeStatus code of the operation

Returns
